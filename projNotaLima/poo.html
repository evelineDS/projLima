Fundamentos da Programação Orientada a Objetos

A unidade curricular Fundamentos de Programação Orientada a Objeto visa propiciar a aquisição de competências básicas relativas a lógica de programação,

como ferramenta para desenvolvimento de aplicações, utilizando linguagem orientada a objetos, bem como o desenvolvimento das competências de gestão adequadas a diferentes situações profissionais
